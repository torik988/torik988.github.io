Comment je passe d'une idée à une prod stable.

01

Capture

Flipper Zero, Raspberry Pi ou laptop + Alfa Network pour sniffer en mobilité.

02

Traitement

Python, Pandas et scripts maison qui normalisent les données et déclenchent des alertes.

03

Visualisation

Dashboards web (Astro/Vite) + animations WebGL pour rendre chaque signal sexy.

04

Automatisation

Docker Compose, CasaOS, GitHub Actions et monitoring custom.
